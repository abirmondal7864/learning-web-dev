Block elements

Block elements are those elements that take up the full width available, and has a line break before and after it.

  • ul, ,ol
  • li
  • p
  • h1 to h6

Inline elements

Inline elements are those elements that only take up as much width as necessary, and does not have a line break before or after it.

  • a tags
  • img
  • strong
